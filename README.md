#  DUT 宿舍门禁自动开门系统
运行在 ESP32-S3-N16R8 上的智能门禁系统，实现从校园网认证到自动开门的完整流程。

---

##  功能特性

-  **统一身份认证** - 自动完成大连理工大学校园网统一登录系统认证
-  **Token 获取** - 自动获取并管理校园统一登录系统 Token
-  **远程开门** - 发送开宿舍门请求，实现无接触开门
-  **WiFi 自动重连** - 校园网环境下稳定的网络连接
-  **流程自动化** - 从联网到开门的一站式自动化处理

## 硬件要求

建议使用内存>2MB版本的单片机

| 组件 | 型号 | 说明 |
|------|------|------|
| 主控芯片 | ESP32-S3-N16R8 | 16MB Flash, 8MB PSRAM |
| 网络 | 2.4GHz WiFi | 连接 DUT 校园网下自己的路由器 |

## ⚠️ 重要配置提醒

使用本项目前，**必须修改以下5处信息**：

| 配置项 | 位置 | 说明 |
|--------|------|------|
| **学号** | 代码开头全局变量 | 大连理工大学统一身份认证学号 |
| **登录密码** | 代码开头全局变量 | 统一登录系统密码 |
| **WiFi名称** | 代码开头全局变量 | ⚠️ **不能是DUT-WLAN校园网**，建议使用个人路由器 |
| **WiFi密码** | 代码开头全局变量 | 对应WiFi的密码 |
| **门锁编号&个人信息** | 约第400行 | 目标宿舍门锁的设备编号&个人学号 |

> 🔴 **注意**：校园网DUT-WLAN有额外的认证限制，无法直接使用，请连接其他可用WiFi（如手机开热点）。
